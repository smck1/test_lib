phaser.evaluation.calc_eer(fpr: ndarray, tpr: ndarray, threshold: ndarray)

Discovers the threshold where FPR and FRR intersects.

Parameters:

  • fpr (np.ndarray) – Array with False Positive Rate from sklearn.metrics.roc_curve

  • tpr (np.ndarray) – Array with True Positive Rate from sklearn.metrics.roc_curve

  • threshold (np.ndarray) – Array with thresholds from sklearn.metrics.roc_curve

Returns:

eer_score, eer_threshold

Return type:

floats, float

Examples

>>> import numpy as np
>>> from sklearn import metrics
>>> y = np.array([1, 1, 2, 2])
>>> scores = np.array([0.1, 0.4, 0.35, 0.8])
>>> fpr, tpr, thresholds = metrics.roc_curve(y, scores, pos_label=2)
>>> eer_score, eer_threshold = calc_eer(fpr, tpr, thresholds)
